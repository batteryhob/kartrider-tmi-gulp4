# kartrider-tmi-gulp4
해당 프로젝트는 scss와 ejs와 sprite image를 구현하기 위한 gulp 4.0 이상 버전의 초기화 프로젝트입니다.  
해당 프로젝트는 다음의 작업을 포함합니다.  
- ejs 빌드
- sass(scss) 빌드
- sprite image
- preview server

- sprite svg 
- 반응형 웹


# Setting
npm install로 포함된 패키지를 설치해줍니다.
- del
- gulp-connect
- gulp-ejs
- gulp-rename
- gulp-sass
- gulp.spritesmith
- node-sass

# Build
$ gulp

# 샘플 최종본입니다.
- 더이상 업데이트되지 않습니다.
